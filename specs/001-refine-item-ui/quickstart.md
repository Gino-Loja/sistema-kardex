# Quickstart: Refine Item UI

## Preconditions

- App running in a browser window.
- User has access to the items list and item detail pages.

## Verify the feature

1. Open the items list view.
2. Confirm there is no thumbnail column in the table.
3. Resize the viewport to 320px-767px and open row actions.
4. Confirm actions are shown inside an overflow menu and do not overlap.
5. Resize to desktop width and confirm actions are visible without the overflow menu.
6. Open an item detail page.
7. Confirm the main image is prominent and fits within the viewport without horizontal scroll.
8. Confirm the existing item fields are shown with stronger visual hierarchy.

## Expected results

- No thumbnails in the items table.
- Action controls are readable and not overlapping at small widths.
- Detail image is large and fully visible on small screens.
- Item detail fields are emphasized without adding new fields.
